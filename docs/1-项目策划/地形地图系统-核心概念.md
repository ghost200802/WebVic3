# 地形地图系统 - 核心概念

## 系统概述

地形地图系统管理游戏世界的地理环境，包括地形类型、资源矿床、地块开发、领土扩张等。地形决定了可建设的建筑和资源分布。

---

## 地形类型

### 基础地形

| 地形 | 特点 | 适用建筑 | 开发难度 |
|------|------|----------|----------|
| 平原 | 地势平坦 | 农田、牧场、工厂 | 低 |
| 山地 | 地势高陡 | 矿场、堡垒 | 高 |
| 森林 | 植被茂密 | 伐木场、牧场 | 中 |
| 水域 | 水面覆盖 | 渔场、港口 | 低 |
| 沙漠 | 干旱少雨 | 特殊建筑 | 高 |
| 冰原 | 寒冷 | 特殊建筑 | 极高 |

---

## 资源矿床

### 矿床类型

| 矿床 | 资源 | 稀有度 | 开采难度 |
|------|------|--------|----------|
| 铁矿床 | 铁矿 | 普通 | 中 |
| 铜矿床 | 铜矿 | 普通 | 中 |
| 煤矿床 | 煤炭 | 普通 | 中 |
| 油田 | 石油 | 稀有 | 高 |
| 铀矿床 | 铀燃料 | 稀有 | 高 |
| 稀土矿床 | 稀土 | 极稀有 | 极高 |

## 地块系统

### 地块属性

```typescript
interface Tile {
  id: string
  x: number
  y: number

  terrain: TerrainType
  resources: ResourceDeposit[]

  development: number
  population: number

  buildings: string[]
  roads: RoadLevel
}
```

### 地块开发

```
开发等级 = Σ(基础设施 × 基础权重) / 最大可能
```

| 开发等级 | 效果 | 建设成本 |
|----------|------|----------|
| 0-20% | 荒地 | 低 |
| 20-40% | 初步开发 | 中 |
| 40-60% | 中等开发 | 中高 |
| 60-80% | 高度开发 | 高 |
| 80-100% | 完全开发 | 极高 |

---

## 系统关系

### 与生产系统

- 地形决定建筑类型
- 矿床提供资源
- 开发等级影响效率

### 与人口系统

- 地形影响人口分布
- 开发等级影响承载力
- 资源影响人口增长

### 与运输系统

- 地形影响运输成本
- 道路建设影响开发
- 距离影响贸易
