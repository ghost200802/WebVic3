# 市场与价格系统 - 总揽

## 系统概述

市场与价格系统模拟复杂的市场经济机制，处理商品定价、供需关系、价格波动、贸易和市场事件。系统通过动态定价机制反映真实市场规律，为玩家提供丰富的经济决策。市场系统是连接生产和消费的桥梁，价格机制是资源配置的核心。

## 核心功能

1. **市场管理**
   - 市场创建和管理
   - 商品上架和下架
   - 价格历史记录
   - 市场统计分析

2. **供需管理**
   - 需求计算和预测
   - 供给计算和预测
   - 供需平衡分析
   - 供需预警

3. **价格管理**
   - 价格形成机制
   - 价格调整机制
   - 价格弹性计算
   - 价格趋势分析

4. **价格波动管理**
   - 波动原因分析
   - 波动幅度控制
   - 波动预测
   - 波动应对

5. **价格干预管理**
   - 干预手段选择
   - 干预效果评估
   - 干预政策调整

6. **贸易管理**
   - 内部贸易管理
   - 外部贸易管理
   - 贸易协定管理
   - 贸易平衡管理

7. **市场事件管理**
   - 事件触发机制
   - 事件效果评估
   - 事件应对策略

## 价格形成机制

```
实际价格 = 基础价格 × 供需系数 × 时代系数 × 市场系数

其中：
- 基础价格 = 生产成本 + 正常利润
- 供需系数 = (1 + 需求/供给)^弹性
- 时代系数：不同时代价格水平不同
- 市场系数：反映市场繁荣程度
```

## 商品类型分类

| 类型 | 说明 | 价格波动 | 需求弹性 | 供给弹性 |
|------|------|----------|----------|----------|
| 原料 | 初级资源 | ±15% | 低 | 低 |
| 中间品 | 加工产品 | ±20% | 中 | 中 |
| 最终品 | 消费品 | ±25% | 高 | 高 |
| 奢侈品 | 高端商品 | ±35% | 很高 | 高 |
| 军用品 | 军事装备 | ±30% | 低 | 中 |

## 系统关联

| 关联系统 | 关联关系 |
|----------|----------|
| 生产系统 | 生产增加供给，供给影响价格，价格引导生产 |
| 人口系统 | 人口影响需求，需求影响价格，价格影响生活水平 |
| 科技系统 | 科技影响生产成本，成本影响价格，价格影响技术选择 |
| 运输系统 | 运输影响贸易成本，成本影响价格，价格影响贸易量 |

## 设计原则

1. **真实性**：市场机制符合经济学规律
2. **平衡性**：供需平衡，价格合理波动
3. **策略性**：价格影响经济决策
4. **互动性**：市场与其他系统相互影响
5. **可预测性**：价格变化有一定规律可循
6. **事件驱动**：市场事件影响价格和供需
