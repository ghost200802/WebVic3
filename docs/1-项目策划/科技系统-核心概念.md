# 科技系统 - 核心概念

## 系统概述

科技系统管理科技研发、科技树、科技扩散、时代晋升等。科技是推动文明进步的核心动力。

---

## 科技树

### 科技分类

| 科技类型 | 研发时间 | 研发成本 | 效果 |
|----------|----------|----------|------|
| 基础科技 | 30-60天 | 低 | 基础能力 |
| 应用科技 | 60-120天 | 中 | 生产提升 |
| 高级科技 | 120-180天 | 高 | 显著提升 |
| 顶级科技 | 180-365天 | 极高 | 革命性提升 |

### 科技依赖

```
科技前置 = 基础科技 + 时代要求
```

---

## 研发系统

### 研发投入

```
研发投入 = 资金 + 科研人员 + 实验设施
```

| 投入类型 | 权重 | 效果 |
|----------|------|------|
| 资金 | 30% | 基础保障 |
| 科研人员 | 40% | 核心动力 |
| 实验设施 | 30% | 加速研发 |

### 研发速度

```
研发速度 = 基础速度 × (1 + 投入加成)
```

---

## 科技扩散

### 扩散机制

```
扩散概率 = 外交关系 × 科技差距 × 扩散系数
```

| 扩散方式 | 扩散速度 | 扩散范围 |
|----------|----------|----------|
| 自主研发 | 慢 | 单一 |
| 科技交流 | 中 | 双向 |
| 技术转移 | 快 | 单向 |
| 情报获取 | 很快 | 单向 |

---

## 时代晋升

### 时代要求

```
时代要求 = 科技数量 + 人口规模 + 经济水平
```

| 时代 | 科技要求 | 人口要求 | 经济要求 |
|------|----------|----------|----------|
| 石器 | 0 | 100 | 低 |
| 青铜 | 5 | 1000 | 中 |
| 铁器 | 10 | 5000 | 中高 |
| 工业 | 20 | 20000 | 高 |
| 电气 | 35 | 50000 | 很高 |
| 信息 | 50 | 100000 | 极高 |
| AI | 70 | 200000 | 超高 |

## 系统关系

### 与生产系统

- 科技解锁新生产方式
- 科技提升生产效率
- 科技降低生产成本

### 与市场系统

- 科技影响商品价格
- 科技创造新需求
- 科技改变市场结构

### 与人口系统

- 科技提高教育水平
- 科技改善生活水平
- 科技影响就业结构
