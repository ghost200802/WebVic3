# 运输系统 - 核心概念

## 系统概述

运输系统管理商品和人员的流动，包括道路系统、运输工具、运力需求等。运输是连接生产和消费的纽带。

---

## 道路系统

### 道路等级

| 道路等级 | 运输能力 | 建设成本 | 维护成本 |
|----------|----------|----------|----------|
| 小径 | 10单位/天 | 低 | 低 |
| 土路 | 50单位/天 | 中 | 中 |
| 碎石路 | 200单位/天 | 中高 | 中 |
| 铺装路 | 500单位/天 | 高 | 中高 |
| 高速路 | 1000单位/天 | 很高 | 高 |

### 道路建设

```
建设时间 = 基础时间 × 地形系数 × 距离
```

| 地形类型 | 地形系数 | 建设难度 |
|----------|----------|----------|
| 平原 | 1.0 | 简单 |
| 山地 | 2.5 | 困难 |
| 森林 | 1.5 | 中等 |
| 水域 | 3.0 | 极困难 |

## 运力需求

### 需求计算

```
运力需求 = 生产运输需求 + 消费运输需求 + 贸易运输需求
```

### 需求类型

| 需求类型 | 占比 | 优先级 | 运输方式 |
|----------|------|--------|----------|
| 生产运输 | 40% | 高 | 陆运为主 |
| 消费运输 | 30% | 中 | 混合运输 |
| 贸易运输 | 20% | 中高 | 水运/陆运 |
| 特殊运输 | 10% | 低 | 专业运输 |

---

## 运力不足影响

### 影响程度

```
影响系数 = min(1, 可用运力 / 需要运力)
```

| 不足程度 | 影响系数 | 生产影响 | 价格影响 |
|----------|----------|----------|----------|
| 轻微 | 0.9-1.0 | 轻微下降 | 轻微上涨 |
| 中等 | 0.7-0.9 | 明显下降 | 明显上涨 |
| 严重 | 0.5-0.7 | 大幅下降 | 大幅上涨 |
| 极度 | <0.5 | 剧烈下降 | 剧烈上涨 |

## 运输成本

### 成本构成

```
运输成本 = 燃料成本 + 人工成本 + 维护成本 + 设备折旧
```

### 成本计算

| 成本类型 | 计算方式 | 占比 |
|----------|----------|------|
| 燃料成本 | 消耗量×燃料价格 | 30% |
| 人工成本 | 工人数量×工资 | 25% |
| 维护成本 | 设备价值×维护率 | 25% |
| 设备折旧 | 设备价值×折旧率 | 20% |

---

## 系统关系

### 与生产系统

- 运力影响生产效率
- 运输成本影响生产成本
- 道路等级影响选址

### 与市场系统

- 运输成本影响商品价格
- 运力影响贸易规模
- 运输效率影响市场响应

### 与地形系统

- 地形影响运输成本
- 道路建设受地形限制
- 运输路线依赖地形

### 与人口系统

- 运输影响人口流动
- 交通便利影响发展
- 运输网络影响城市布局
