# 人口与就业系统 - 核心概念

## 系统概述

人口与就业系统管理城市人口、就业、生活水平、社会流动等核心社会指标。人口是生产和消费的基础，就业是经济的核心，生活水平反映社会福祉。

---

## 人口系统

### 人口构成

#### 年龄结构

```
总人口 = 幼儿 + 青少年 + 成年人 + 老年人
```

| 年龄段 | 占比 | 经济活动 | 消费特点 |
|--------|------|----------|----------|
| 0-14岁 | 20-25% | 非劳动人口 | 教育、医疗需求高 |
| 15-64岁 | 60-70% | 劳动人口 | 生产、消费主力 |
| 65岁+ | 10-15% | 退休人口 | 医疗需求高 |

#### 性别结构

```
性别比 = 男性人口 / 女性人口
```

| 性别比 | 状态 | 影响 |
|--------|------|------|
| 0.95-1.05 | 平衡 | 正常发展 |
| 1.05-1.10 | 轻微失衡 | 影响婚配 |
| 1.10-1.20 | 明显失衡 | 社会问题 |
| >1.20 | 严重失衡 | 严重社会问题 |

## 就业系统

### 就业类型

#### 生产就业

直接参与生产活动的就业。

| 行业类型 | 占比 | 技能要求 | 收入水平 |
|----------|------|----------|----------|
| 农业 | 20-40% | 低 | 低 |
| 工业 | 15-35% | 中 | 中 |
| 服务业 | 25-50% | 中高 | 中高 |
| 高科技 | 5-15% | 高 | 高 |

#### 服务业就业

提供服务活动的就业。

| 服务类型 | 占比 | 技能要求 | 收入水平 |
|----------|------|----------|----------|
| 基础服务 | 10-20% | 低 | 低 |
| 商业服务 | 15-25% | 中 | 中 |
| 专业服务 | 10-20% | 高 | 高 |
| 创意服务 | 5-10% | 中高 | 中高 |

## 生活水平

### 生活水平指标

#### 基础指标

| 指标 | 权重 | 测量方式 | 目标值 |
|------|------|----------|--------|
| 粮食保障 | 30% | 粮食消费/需求 | ≥100% |
| 住房条件 | 25% | 人均住房面积 | ≥15㎡ |
| 医疗保障 | 20% | 医疗服务覆盖 | ≥90% |
| 教育水平 | 15% | 受教育程度 | ≥高中 |
| 休闲娱乐 | 10% | 休闲时间 | ≥2小时/天 |

#### 生活水平计算

```
生活水平 = Σ(指标分值 × 指标权重)
```

| 生活水平 | 分值 | 状态 |
|----------|------|------|
| 0-20 | 0-20 | 贫困 |
| 20-40 | 20-40 | 困难 |
| 40-60 | 40-60 | 正常 |
| 60-80 | 60-80 | 良好 |
| 80-100 | 80-100 | 富裕 |

## 系统关系

### 与生产系统

- 人口提供劳动力
- 就业决定生产规模
- 生活水平影响工作效率

### 与市场系统

- 人口影响需求
- 就业影响收入
- 生活水平影响消费结构

### 与科技系统

- 教育影响科技研发
- 生活水平影响科技需求
- 科技影响就业结构

### 与地形系统

- 人口密度影响土地需求
- 地形影响人口分布
- 就业影响城市发展

### 人口管理器接口

```typescript
interface IPopulationManager {
  getPopulation(): number
  getPopulationByAge(): AgeDistribution

  getEmploymentRate(): number
  getUnemploymentRate(): number

  children: number
```
