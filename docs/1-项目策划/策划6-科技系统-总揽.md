# 科技系统 - 总揽

## 系统概述

科技系统管理科技研发、科技树、科技扩散、时代晋升等。科技是推动文明进步的核心动力，影响生产效率、军事力量、生活水平和社会发展。科技系统通过研发、扩散和应用，推动文明从石器时代向AI时代演进。

## 核心功能

1. **科技树管理**
   - 科技分类和组织
   - 科技依赖关系
   - 科技解锁机制
   - 科技树可视化

2. **研发管理**
   - 研发投入分配
   - 研发速度计算
   - 研发优先级设置
   - 研发进度追踪

3. **科技扩散管理**
   - 扩散机制控制
   - 扩散概率计算
   - 扩散范围管理
   - 扩散效果评估

4. **科技应用管理**
   - 应用领域管理
   - 应用效果评估
   - 应用优化

5. **时代管理**
   - 时代定义
   - 时代要求检查
   - 时代晋升触发
   - 时代特性应用

## 科技分类

| 科技类型 | 研发时间 | 研发成本 | 效果 | 示例 |
|----------|----------|----------|------|------|
| 基础科技 | 30-60天 | 低 | 基础能力 | 石器制作、火的使用 |
| 应用科技 | 60-120天 | 中 | 生产提升 | 农耕技术、冶金技术 |
| 高级科技 | 120-180天 | 高 | 显著提升 | 蒸汽机、电力技术 |
| 顶级科技 | 180-365天 | 极高 | 革命性提升 | 核聚变、人工智能 |

## 科技依赖关系

```
科技前置 = 基础科技 + 时代要求

依赖类型：
- 强依赖：必须完成前置科技
- 弱依赖：前置科技提供加成
- 时代依赖：达到特定时代才能解锁
- 条件依赖：满足特定条件才能解锁
```

## 设计原则

1. **真实性**：科技发展符合历史规律
2. **平衡性**：科技投入与收益相匹配
3. **策略性**：科技选择影响发展路径
4. **演进性**：科技随时代逐步发展
5. **互动性**：科技与其他系统相互影响
6. **扩散性**：科技可通过多种方式传播
