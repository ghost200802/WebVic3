# 生产系统 - 生产方式演进

## 概述

生产方式代表不同的技术水平和生产模式。随着时代进步，新的生产方式不断涌现，大幅提升生产效率。本文档描述各类建筑的生产方式演进过程。

---

### 生产方式详细说明

#### 刀耕火种
- 产出：谷物 10单位/天
- 投入：无
- 特点：原始耕作，土地需休耕
- 缺点：效率低下，环境影响大

#### 畜力耕作
- 产出：谷物 15单位/天
- 投入：饲料 2单位/天
- 特点：使用牲畜拉犁
- 优点：效率提升1.5倍

#### 铁犁牛耕
- 产出：谷物 20单位/天
- 投入：饲料 3单位/天，铁器维护
- 特点：铁制农具
- 优点：效率翻倍

#### 轮作制
- 产出：谷物 25单位/天
- 投入：饲料 4单位/天
- 特点：作物轮作，土地永续利用
- 优点：土地利用率高

#### 农业革命
- 产出：谷物 40单位/天
- 投入：饲料 6单位/天，化肥 2单位/天
- 特点：新作物品种，化肥使用
- 优点：产量大幅提升

#### 机械化
- 产出：谷物 80单位/天
- 投入：饲料 8单位/天，化肥 3单位/天，燃油 2单位/天
- 特点：拖拉机、收割机
- 优点：自动化程度高

#### 现代农业
- 产出：谷物 120单位/天
- 投入：饲料 10单位/天，化肥 5单位/天，电力 3单位/天
- 特点：化肥、农药、转基因作物
- 优点：产量稳定

#### 智能农业
- 产出：谷物 200单位/天
- 投入：化肥 6单位/天，电力 5单位/天
- 特点：AI管理，精准农业
- 优点：完全自动化

- 特点：工匠个体生产

- 特点：工匠分工协作

- 优点：规模化效应

- 特点：标准化流水线生产

- 特点：机器人辅助生产
- 优点：减少人工
- 缺点：设备成本高

#### 智能制造
- 特点：全自动化智能生产
- 优点：最高效率
- 缺点：高技术门槛

## 生产方式切换机制

### 切换规则

1. **解锁条件**：必须满足科技或时代要求
2. **切换成本**：无成本，可随时切换
3. **生效时间**：切换后立即生效
4. **工人影响**：工人需要时间适应新方式（效率逐步提升）

### 切换过渡期

切换生产方式时，存在过渡期：

```
过渡期天数 = 7 × (新旧方式效率差 / 新方式效率)
```

过渡期内效率逐步提升：
- 第1天：50%
- 第3天：75%
- 第5天：90%
- 第7天：100%

### 投入品变化

切换生产方式可能导致投入品变化：

1. 检查新方式所需投入品
2. 若新投入品库存不足，效率下降
3. 效率 = min(1, 库存充足率)

## 生产方式选择策略

### 策略考虑因素

#### 1. 投入品可用性
- 高效生产方式可能需要复杂投入品
- 投入品不足时，高效方式反而降低效率
- 需确保供应链稳定

#### 2. 工人素质
- 高效生产方式需要更高教育水平工人
- 工人技能不足时，效率无法充分发挥
- 需要配套教育设施

#### 3. 成本效益
- 高效方式通常需要更高运营成本
- 需计算投入产出比
- 选择最经济的方式

#### 4. 产品需求
- 根据市场需求调整生产方式
- 高端产品需要高效方式
- 低端产品可使用低成本方式

### 推荐策略

#### 早期（石器-铁器）
- 使用基础生产方式
- 优先解锁新生产方式科技
- 积累资源和人口

#### 中期（中世纪-工业）
- 逐步升级生产方式
- 建立完整供应链
- 培养高素质工人

#### 后期（电气-AI）
- 全面采用高效生产方式
- 实现全自动化
- 追求最高效率
