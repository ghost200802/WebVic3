# 项目总揽

## 项目概述

WebVic3 是一款文明模拟游戏，玩家从石器时代开始，通过发展经济、科技、军事，逐步推进到 AI 时代。游戏以复杂的经济系统为核心，包含生产、市场、人口、运输、科技、地形等多个相互关联的系统。

## 核心设计理念

### 1. 系统关联性

所有系统相互关联，形成复杂的互动网络：
- 地形影响建筑和生产
- 人口提供劳动力，影响需求
- 生产提供商品，增加供给
- 市场通过价格调节供需
- 运输连接生产和消费
- 科技推动时代进步

### 2. 时代演进

游戏分为 9 个时代，每个时代有独特的特征：

| 时代 | 核心特征 | 关键技术 |
|------|----------|----------|
| 石器 | 采集狩猎 | 石器制作 |
| 青铜 | 农业手工业 | 冶金技术 |
| 铁器 | 封建社会 | 铁器技术 |
| 古典 | 贸易繁荣 | 城市建设 |
| 中世纪 | 庄园经济 | 封建制度 |
| 工业 | 工业化 | 蒸汽机 |
| 电气 | 电气化 | 电力技术 |
| 信息 | 信息化 | 计算机 |
| AI | 智能化 | 人工智能 |

### 3. 经济循环

```
人口 → 需求 → 市场价格 → 生产决策 → 商品产出 → 消费 → 生活水平 → 人口增长
```

### 4. 战略深度

玩家需要在多个维度做出决策：
- 生产方式选择：不同方式有不同效率和成本
- 技术研发路径：优先发展哪个领域
- 资源配置：劳动力、资金、资源的分配
- 市场策略：价格干预、贸易政策
- 基础设施建设：道路、城市布局

## 游戏目标

1. **经济繁荣**：提高生活水平，增加财富
2. **科技领先**：率先进入新时代
3. **人口增长**：吸引更多人口，提高就业
4. **领土扩张**：开发和占领更多地块
5. **文明成就**：解锁所有科技，建立伟大文明

## 开发原则

1. **真实性**：模拟真实的经济规律和社会发展
2. **平衡性**：避免单一最优策略，鼓励多样性
3. **可扩展性**：支持新增系统和内容
4. **性能优化**：支持大规模模拟
5. **易学难精**：简单易懂，深度可控
