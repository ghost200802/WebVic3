# 生产系统 - 建筑类型

## 概述

建筑是进行生产的场所，不同类型的建筑提供不同的生产功能。本文档详细描述各类建筑的特点、用途和时代要求。

**核心设计原则：**
- 所有建筑统一占用 **1单位土地**
- 所有建筑基础配置 **10个工人**
- 资源计量标准：**1单位 = 100公斤**
- 建筑通过 **count（数量）** 进行堆叠，可自由建造任意数量
- 建筑数量仅受土地资源限制，无固定上下限
- 产能基于可靠的历史文献数据换算，体现真实的生产力发展

## 建筑堆叠系统

### 堆叠规则

- **同一类型的建筑可以堆叠显示**
- 堆叠建筑的 `count` 表示该类型建筑的总数量
- 堆叠建筑的 `workers` 是所有子建筑的工人总和
- 堆叠建筑的 `baseWorkers` = 单个建筑的 `baseWorkers` × `count`
- 堆叠建筑的 `maxWorkers` = 单个建筑的 `maxWorkers` × `count`
- 堆叠建筑的 `baseThroughput` = 单个建筑的 `baseThroughput` × `count`

### 堆叠计算

```
总工位数 = 基础工位数 × 数量
最大工位数 = 最大工位数 × 数量
基础产能 = 单个建筑基础产能 × 数量
实际产能 = 基础产能 × 效率
效率 = min(1, 实际工人数 / 总工位数)
```

### 建筑数量限制

- **建筑数量无固定上限**
- 仅受土地资源限制
- 玩家可根据需要在可用土地上建造任意数量的同类型建筑
- 每个建筑占用1单位土地

## 建筑建造规则

### 土地占用

- **所有建筑统一占用1单位土地**
- 1单位土地面积约等于1000平方米（约1.5市亩）

### 地形要求

| 建筑类别 | 可用地形 | 特殊要求 |
|----------|----------|----------|
| 资源建筑 | 根据类型而定 | 矿场需对应矿床 |
| 工业建筑 | 平原 | 需道路 |
| 公用建筑 | 任意 | 需道路 |
| 能源建筑 | 平原/山地/水域 | 根据类型而定 |

### 邻接加成

某些建筑相邻时提供加成：

| 建筑组合 | 加成效果 |
|----------|----------|
| 磨坊 + 农田 | 磨坊效率+10% |
| 纺织厂 + 染坊 | 染坊效率+15% |
| 钢铁厂 + 煤矿 | 钢铁厂效率+10% |
| 发电厂 + 水源 | 发电厂效率+15% |
| 市场 + 仓库 | 交易效率+10% |

### 运输要求

建筑需要道路连接才能正常生产：

```
运输效率 = min(1, 可用运力 / 所需运力)
```

运力不足时，建筑效率下降。

## 建筑维护

### 维护成本

建筑需要定期维护，否则效率下降：

```
维护成本 = 基础维护成本 × 建筑数量
```

### 维护不足的影响

| 维护度 | 效率 |
|--------|------|
| 100% | 100% |
| 80-99% | 90% |
| 60-79% | 70% |
| 40-59% | 40% |
| 20-39% | 10% |
| <20% | 0% |

### 维护方法

- 自动维护：消耗维护成本，保持100%
- 手动维护：不消耗成本，效率下降

---

## 建筑拆除

### 拆除成本

```
拆除成本 = 0.5 × 单个建筑价值 × 数量
拆除返还 = 0.3 × 单个建筑建造成本 × 数量
```

### 拆除时间

```
拆除时间 = 0.5 × 单个建筑建造时间
```

### 拆除限制

- 正在生产的建筑不能立即拆除
- 关键基础设施需确认
- 防止误操作

## 产能数据说明

### 产能换算原则

所有产能数据基于可靠的历史文献和考古研究，按照以下标准换算：

**土地单位换算：**
- 游戏中1单位土地 = 1000平方米 ≈ 1.5市亩
- 1市亩 ≈ 666.7平方米
- 1公顷 = 15市亩 = 10000平方米

**工人耕种能力（历史数据）：**
- 古代人均可耕种约1.5市亩/人
- 10个工人约可耕种15市亩（约10,000平方米）

### 中国历代粮食亩产数据（历史文献）

根据《中国历代粮食亩产变化一览表》等文献资料：

| 朝代 | 亩产（公斤/市亩） | 折算日产（15市亩） |

| 汉朝 | 95公斤/亩 | 3.9公斤/天 |

### 各时代建筑产能数据来源

#### 石器时代（采集狩猎）
- 数据来源：考古学研究，原始采集狩猎活动
- 10人日产：2公斤，勉强维持生存

#### 青铜时代（秦汉水平）
- 数据来源：历史文献记载，秦汉时期亩产65-95公斤
- 计算公式：65-95公斤/亩 × 15亩 ÷ 365天 = 2.7-3.9公斤/天

#### 铁器时代（隋唐水平）
- 数据来源：《新唐书·食货志》记载，隋唐时期亩产约138公斤
- 计算公式：138公斤/亩 × 15亩 ÷ 365天 = 5.7公斤/天

#### 中世纪（宋代水平）
- 数据来源：范仲淹《答手诏陈十事》，宋代亩产约225公斤
- 计算公式：225公斤/亩 × 15亩 ÷ 365天 = 9.2公斤/天

#### 工业时代（机械化初期）
- 数据来源：1949年全国平均亩产68.6公斤，1950年南方水稻282斤（141公斤）
- 机械化后亩产提升至约300-400公斤
- 计算公式：300-400公斤/亩 × 15亩 ÷ 365天 = 12-16公斤/天

#### 电气时代（工业化高峰）
- 数据来源：现代工业化农业产量
- 亩产可达400-500公斤
- 计算公式：400-500公斤/亩 × 15亩 ÷ 365天 = 16-21公斤/天

#### 信息时代（现代农业）
- 数据来源：袁隆平超级杂交水稻试验数据，亩产1000公斤
- 两季稻亩产1500公斤
- 计算公式：1000-1500公斤/亩 × 15亩 ÷ 365天 = 41-62公斤/天

#### AI时代（智能农业）
- 数据来源：未来农业发展趋势预测
- 全自动化智能农业亩产可达1600-2000公斤
- 计算公式：1600-2000公斤/亩 × 15亩 ÷ 365天 = 66-82公斤/天

## 设计说明

### 统一化设计

1. **土地占用统一**：所有建筑占用1单位土地（约1000平方米），简化规划
2. **工人数量统一**：基础配置10个工人，便于平衡
3. **资源单位统一**：1单位=100公斤，易于理解和计算

### 分类清晰

1. **资源建筑**：获取原材料，依赖自然资源
2. **工业建筑**：加工原材料，生产成品
3. **公用建筑**：提供基础设施和服务
4. **能源建筑**：生产能源，支持其他建筑运行

### 堆叠系统

1. **无上限限制**：建筑数量无固定上限，仅受土地限制
2. **灵活扩展**：可根据需要自由建造任意数量的同类型建筑
3. **堆叠显示**：同类型建筑在UI中合并显示，通过数量标识
4. **线性增长**：产能与建筑数量呈线性关系

### 平衡性设计

1. 时代递进：随着时代发展，建筑产能显著提升
2. 成本收益：早期建筑成本低但产能低，后期成本高但产能高
3. 地形限制：某些建筑只能建造在特定地形上
4. 邻接加成：合理布局可获得额外效率加成
5. **历史真实性**：产能数据基于可靠的历史文献和考古研究，体现真实的生产力发展历程
