# 生产系统 - 建筑类型

## 概述

建筑是进行生产的场所，不同类型的建筑提供不同的生产功能。本文档详细描述各类建筑的特点、用途和时代要求。

## 建筑属性详解

### 基础属性

```typescript
interface Building {
  id: string
  name: string
  type: BuildingType

  minEra: Era
  maxEra?: Era

  constructionCost: Record<string, number>
  constructionTime: number

  baseWorkers: number
  maxWorkers: number
  currentWorkers: number

  baseThroughput: number

  productionMethods: string[]
  level: number
  experience: number
  tileId: string
}
```

### 建筑等级

每个建筑可升级多个等级，提升各项属性：

| 等级 | 产能加成 | 工位数加成 | 升级成本 |
|------|----------|------------|----------|
| 1 | 100% | 100% | 基础成本 |
| 2 | 120% | 110% | 1.5x基础成本 |
| 3 | 140% | 120% | 2.0x基础成本 |
| 4 | 165% | 130% | 2.5x基础成本 |
| 5 | 195% | 145% | 3.0x基础成本 |
| 6 | 230% | 160% | 3.5x基础成本 |
| 7 | 270% | 175% | 4.0x基础成本 |
| 8 | 315% | 190% | 4.5x基础成本 |
| 9 | 365% | 205% | 5.0x基础成本 |
| 10 | 420% | 220% | 5.5x基础成本 |

### 建筑经验

建筑通过日常生产获得经验，经验用于升级：

```
每日经验 = 基础经验 × 产出效率
```

| 产出效率 | 每日经验 |
|----------|----------|
| <50% | 1 |
| 50-80% | 2 |
| 80-100% | 3 |
| 100-150% | 5 |
| >150% | 8 |

### 升级要求

```
升级所需经验 = 基础经验 × 建筑等级^1.5
```

| 建筑等级 | 所需经验 |
|----------|----------|
| 1→2 | 100 |
| 2→3 | 283 |
| 3→4 | 580 |
| 4→5 | 1000 |
| 5→6 | 1565 |
| 6→7 | 2290 |
| 7→8 | 3196 |
| 8→9 | 4290 |
| 9→10 | 5590 |

## 建筑建造规则

### 地形要求

| 建筑类型 | 可用地形 | 特殊要求 |
|----------|----------|----------|
| 农田 | 平原 | 无 |
| 牧场 | 平原 | 无 |
| 矿场 | 山地 | 需对应矿床 |
| 渔场 | 水域 | 无 |
| 伐木场 | 森林 | 无 |
| 工厂 | 平原 | 需道路 |
| 发电厂 | 平原/山地 | 需道路 |
| 仓库 | 任意 | 需道路 |

### 邻接加成

某些建筑相邻时提供加成：

| 建筑组合 | 加成效果 |
|----------|----------|
| 磨坊 + 农田 | 磨坊效率+10% |
| 纺织厂 + 染坊 | 染坊效率+15% |
| 钢铁厂 + 煤矿 | 钢铁厂效率+10% |
| 发电厂 + 水源 | 发电厂效率+15% |

### 运输要求

建筑需要道路连接才能正常生产：

```
运输效率 = min(1, 可用运力 / 所需运力)
```

运力不足时，建筑效率下降。

## 建筑维护

### 维护成本

建筑需要定期维护，否则效率下降：

```
维护成本 = 基础维护成本 × 建筑等级
```

### 维护不足的影响

| 维护度 | 效率 |
|--------|------|
| 100% | 100% |
| 80-99% | 90% |
| 60-79% | 70% |
| 40-59% | 40% |
| 20-39% | 10% |
| <20% | 0% |

### 维护方法

- 自动维护：消耗维护成本，保持100%
- 手动维护：不消耗成本，效率下降

---

## 建筑拆除

### 拆除成本

```
拆除成本 = 0.5 × 当前建筑价值
拆除返还 = 0.3 × 建造成本
```

### 拆除时间

```
拆除时间 = 0.5 × 建造时间
```

### 拆除限制

- 正在生产的建筑不能立即拆除
- 关键基础设施需确认
- 防止误操作
