# 市场与价格系统 - 价格干预

## 概述

价格干预是政府或玩家对市场价格进行调控的手段，包括价格上限、价格下限、补贴政策、税收政策等。价格干预可以稳定市场，但也可能带来副作用。

---

## 干预类型

### 价格上限

设定商品的最高价格。

#### 上限设定

```
价格上限 = 基础价格 × 上限系数
```

| 上限系数 | 价格水平 | 目的 |
|----------|----------|------|
| 0.8 | 低于市场 | 保护消费者 |
| 0.9 | 略低于市场 | 轻微干预 |
| 1.0 | 市场价格 | 无干预 |
| 1.1 | 高于市场 | 无实际作用 |

#### 适用场景

| 场景 | 理由 | 预期效果 |
|------|------|----------|
| 粮食短缺 | 保障基本需求 | 稳定社会 |
| 灾害期间 | 防止投机 | 控制价格 |
| 通胀高企 | 抑制价格过快上涨 | 稳定经济 |

#### 效果分析

| 效果类型 | 短期效果 | 长期效果 |
|----------|----------|----------|
| 价格稳定 | ✅ 稳定价格 | ⚠️ 可能导致短缺 |
| 供给充足 | ✅ 价格合理 | ❌ 供给减少 |
| 消费者福利 | ✅ 价格降低 | ⚠️ 可能购买不到 |

#### 副作用

| 副作用 | 严重程度 | 应对措施 |
|--------|----------|----------|
| 供给不足 | 高 | 增加供给 |
| 黑市交易 | 中 | 加强监管 |
| 质量下降 | 中 | 质量检查 |
| 投资不足 | 高 | 长期补贴 |

```

```

| 0.9 | 略低于市场 | 无实际作用 |

| 1.1 | 高于市场 | 保护生产者 |
| 1.2 | 明显高于市场 | 强力保护 |

#### 适用场景

| 场景 | 理由 | 预期效果 |
|------|------|----------|
| 农产品 | 保护农民收入 | 稳定生产 |
| 工业品 | 防止恶性竞争 | 保护产业 |

#### 效果分析

| 效果类型 | 短期效果 | 长期效果 |
|----------|----------|----------|
| 价格稳定 | ✅ 稳定价格 | ⚠️ 可能导致过剩 |

#### 副作用

| 副作用 | 严重程度 | 应对措施 |

| 财政负担 | 高 | 控制规模 |

## 补贴政策

对生产或消费提供资金支持。

#### 补贴类型

| 补贴类型 | 受益对象 | 补贴方式 | 补贴金额 |
|----------|----------|----------|----------|
| 生产补贴 | 生产者 | 直接补贴 | 10-20% |
| 消费补贴 | 消费者 | 价格优惠 | 5-10% |
| 出口补贴 | 出口商 | 退税/补贴 | 15-25% |
| 进口补贴 | 进口商 | 关税减免 | 10-20% |

#### 生产补贴

```
补贴后价格 = 市场价格 - 补贴金额
```

| 补贴金额 | 效果 | 持续时间 |
|----------|------|----------|
| 5% | 轻微降低价格 | 长期 |
| 10% | 中等降低价格 | 长期 |
| 15% | 明显降低价格 | 长期 |
| 20% | 大幅降低价格 | 长期 |

#### 消费补贴

```
补贴后价格 = 市场价格 × (1 - 补贴率)
```

| 补贴率 | 效果 | 适用对象 |
|--------|------|----------|
| 5% | 轻微优惠 | 大众商品 |
| 8% | 中等优惠 | 必需商品 |
| 10% | 明显优惠 | 特定群体 |

#### 出口补贴

```
出口补贴 = 出口额 × 补贴率
```

| 补贴率 | 效果 | 贸易影响 |
|--------|------|----------|
| 10% | 轻微促进 | 轻微扭曲 |
| 15% | 中等促进 | 中等扭曲 |
| 20% | 明显促进 | 明显扭曲 |
| 25% | 大幅促进 | 严重扭曲 |

#### 效果分析

| 效果类型 | 短期效果 | 长期效果 |
|----------|----------|----------|
| 价格降低 | ✅ 价格下降 | ⚠️ 财政负担 |
| 需求增加 | ✅ 需求上升 | ⚠️ 可能过度消费 |
| 生产扩大 | ✅ 生产增加 | ⚠️ 效率下降 |

#### 副作用

| 副作用 | 严重程度 | 应对措施 |
|--------|----------|----------|
| 财政负担 | 高 | 控制规模 |
| 效率低下 | 中 | 定期评估 |
| 市场扭曲 | 高 | 逐步取消 |
| 贪污腐败 | 中 | 加强监管 |

## 税收政策

对商品交易征收税款。

#### 税收类型

| 税收类型 | 征收对象 | 税收方式 | 税收税率 |
|----------|----------|----------|----------|
| 消费税 | 消费者 | 价格加成 | 5-15% |
| 增值税 | 交易双方 | 增值征收 | 10-20% |
| 奢侈税 | 奢侈品 | 高额征税 | 20-40% |
| 资源税 | 资源开采 | 开采征税 | 5-15% |

#### 消费税

```
税收后价格 = 市场价格 × (1 + 税收税率)
```

| 税收税率 | 效果 | 适用商品 |
|----------|------|----------|
| 5% | 轻微加价 | 普通商品 |
| 8% | 中等加价 | 中档商品 |
| 10% | 明显加价 | 高档商品 |
| 15% | 大幅加价 | 奢侈商品 |

#### 增值税

```
增值税 = (销售额 - 进项) × 增值税率
```

| 增值税率 | 效果 | 适用场景 |
|----------|------|----------|
| 10% | 低税收 | 刺激经济 |
| 13% | 中税收 | 标准税收 |
| 15% | 高税收 | 正常税收 |
| 20% | 很高税收 | 财政收入 |

#### 奢侈税

```
奢侈税 = 商品价格 × 奢侈税率
```

| 奢侈税率 | 效果 | 适用商品 |
|----------|------|----------|
| 20% | 中等征税 | 一般奢侈 |
| 30% | 高额征税 | 高档奢侈 |
| 40% | 极高征税 | 顶级奢侈 |

#### 资源税

```
资源税 = 开采价值 × 资源税率
```

| 资源税率 | 效果 | 适用资源 |
|----------|------|----------|
| 5% | 轻微征税 | 普通资源 |
| 10% | 中等征税 | 重要资源 |
| 15% | 高额征税 | 战略资源 |

#### 效果分析

| 效果类型 | 短期效果 | 长期效果 |
|----------|----------|----------|
| 价格上涨 | ❌ 价格上升 | ✅ 财政收入 |
| 需求下降 | ⚠️ 需求减少 | ⚠️ 消费调整 |
| 生产调整 | ⚠️ 生产调整 | ✅ 资源优化 |

#### 副作用

| 副作用 | 严重程度 | 应对措施 |
|--------|----------|----------|
| 消费负担 | 中 | 对必需品减免 |
| 生产下降 | 中 | 对生产环节减税 |
| 偷税漏税 | 高 | 加强征管 |
| 贸易扭曲 | 高 | 协调税收政策 |

## 关税政策

对进口商品征收关税。

#### 关税类型

| 关税类型 | 税收税率 | 适用商品 | 目的 |
|----------|----------|----------|------|
| 普通关税 | 5-15% | 大部分商品 | 财政收入 |
| 保护关税 | 15-35% | 特定产业 | 产业保护 |
| 惩罚关税 | 35-50% | 特定国家 | 政治施压 |
| 禁止关税 | 100% | 特定商品 | 贸易禁运 |

#### 关税效果

```
关税后价格 = 进口价格 × (1 + 关税税率)
```

| 关税税率 | 价格上涨 | 进口减少 | 国产增加 |
|----------|----------|----------|----------|
| 5% | +5% | -5% | +3% |
| 10% | +10% | -10% | +7% |
| 15% | +15% | -15% | +12% |
| 25% | +25% | -25% | +20% |
| 35% | +35% | -35% | +28% |
| 50% | +50% | -50% | +40% |

#### 关税影响

| 影响类型 | 短期效果 | 长期效果 |
|----------|----------|----------|
| 价格影响 | ❌ 价格上升 | ⚠️ 部分转嫁 |
| 产业保护 | ✅ 保护产业 | ⚠️ 可能效率低下 |
| 消费者 | ❌ 增加负担 | ⚠️ 选择减少 |
| 国际关系 | ⚠️ 可能恶化 | ⚠️ 贸易摩擦 |

## 配额制度

限制商品交易数量。

#### 配额类型

| 配额类型 | 限制方式 | 限制程度 | 适用场景 |
|----------|----------|----------|----------|
| 进口配额 | 限制进口量 | 中等 | 保护产业 |
| 出口配额 | 限制出口量 | 中等 | 确保供应 |
| 生产配额 | 限制生产量 | 中等 | 控制产量 |

#### 配额效果

```
实际贸易量 = min(配额数量, 需求量)
```

| 配额比例 | 贸易减少 | 价格影响 | 生产影响 |
|----------|----------|----------|----------|
| 90% | -10% | +5% | +3% |
| 80% | -20% | +10% | +7% |
| 70% | -30% | +15% | +12% |
| 50% | -50% | +30% | +25% |
| 30% | -70% | +50% | +40% |

## 价格管制

直接控制商品价格。

#### 管制类型

| 管制类型 | 管制方式 | 适用商品 | 目的 |
|----------|----------|----------|------|
| 完全管制 | 政府定价 | 战略商品 | 确保供应 |
| 部分管制 | 限制波动 | 重要商品 | 稳定价格 |
| 临时管制 | 紧急干预 | 应急商品 | 应对危机 |

#### 管制效果

| 管制类型 | 价格稳定 | 供应保障 | 市场效率 |
|----------|----------|----------|----------|
| 完全管制 | ✅ 完全稳定 | ✅ 保障供应 | ❌ 效率低下 |
| 部分管制 | ✅ 相对稳定 | ⚠️ 部分保障 | ⚠️ 效率下降 |
| 临时管制 | ✅ 短期稳定 | ⚠️ 应急保障 | ❌ 短期影响 |

## 干预组合

### 常见组合

| 组合类型 | 干预措施 | 适用场景 | 效果 |
|----------|----------|----------|------|
| 上限+补贴 | 价格上限+生产补贴 | 粮食短缺 | 保障供应 |
| 下限+收购 | 价格下限+政府收购 | 农产品 | 保护农民 |
| 关税+补贴 | 进口关税+出口补贴 | 产业保护 | 双向保护 |
| 税收+补贴 | 高税收+定向补贴 | 奢侈品 | 调节消费 |

### 组合效果

```
组合效果 = Σ(各干预措施效果 × 组合系数)
```

| 干预措施数量 | 组合系数 | 说明 |
|--------------|----------|------|
| 1个 | 1.0 | 完全效果 |
| 2个 | 0.9 | 略微减弱 |
| 3个 | 0.8 | 明显减弱 |
| 4个+ | 0.7 | 显著减弱 |

## 干预成本

### 财政成本

```
财政成本 = 补贴支出 + 收购支出 + 管理成本 - 税收收入
```

| 干预类型 | 财政成本 | 成本构成 |
|----------|----------|----------|
| 补贴政策 | 高 | 补贴支出 |
| 收购政策 | 高 | 收购支出 |
| 价格管制 | 中 | 管理成本 |
| 税收政策 | 负 | 税收收入 |

### 经济成本

```
经济成本 = 效率损失 + 市场扭曲 + 资源错配
```

| 干预类型 | 经济成本 | 成本构成 |
|----------|----------|----------|
| 价格管制 | 高 | 效率损失 |
| 配额制度 | 高 | 资源错配 |
| 关税政策 | 中 | 市场扭曲 |
| 税收政策 | 低 | 小幅扭曲 |

## 干预系统接口定义

### 干预管理器接口

```typescript
interface IInterventionManager {
  setPriceCeiling(goodsId: string, price: number): void
  setPriceFloor(goodsId: string, price: number): void

  setSubsidy(goodsId: string, type: SubsidyType, amount: number): void
  setTax(goodsId: string, type: TaxType, rate: number): void

  setTariff(goodsId: string, rate: number): void
  setQuota(goodsId: string, quota: number): void

  evaluateIntervention(interventionId: string): InterventionResult

  getInterventionHistory(): InterventionRecord[]
}

interface InterventionResult {
  priceStability: number
  supplySecurity: number
  productionProtection: number
  fiscalSustainability: number
  overallScore: number
}
```
